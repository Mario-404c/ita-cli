# ita-cli

Client CLI in italiano per lo streaming di anime, ispirato ad [ani-cli](https://github.com/pystardust/ani-cli).

## Requisiti
- Python 3.x
- mpv oppure ffplay (ffmpeg) installati ed aggiunti al PATH

## Dipendenze Python
pip install requests beautifulsoup4 rich questionary

## Utilizzo
python script.py
